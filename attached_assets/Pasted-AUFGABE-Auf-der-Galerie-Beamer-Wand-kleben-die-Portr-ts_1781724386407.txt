AUFGABE
Auf der Galerie-/Beamer-Wand kleben die Porträts am linken Rand, wenn nur wenige da
sind — rechts bleibt großer leerer Raum. Fix: Wenn alle Porträts zusammen schmaler
sind als die Wand, sollen sie horizontal ZENTRIERT werden. Überläuft die Wand (viele
Profile), bleibt alles exakt wie bisher. Zwei kleine Ergänzungen in EINER Datei.

DATEI
artifacts/emmerich/src/pages/Galerie.tsx

ÄNDERUNG 1 — Zentrier-Versatz berechnen
Suche EXAKT diesen Block:

  // Windowing
  const containerW   = containerRef.current?.clientWidth ?? 380;

Ersetze ihn EXAKT durch:

  // Windowing
  const containerW   = containerRef.current?.clientWidth ?? 380;
  const centerShift  = Math.max(0, (containerW - totalWidth) / 2);

ÄNDERUNG 2 — Versatz auf die Porträt-Reihe anwenden
Suche EXAKT diesen Block:

        {/* Porträt-Reihe */}
        <div style={{ position: "absolute", inset: 0, zIndex: 2 }}>

Ersetze ihn EXAKT durch:

        {/* Porträt-Reihe */}
        <div style={{ position: "absolute", inset: 0, zIndex: 2, transform: `translateX(${centerShift}px)` }}>
