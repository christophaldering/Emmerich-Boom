## 4. DESIGN-SYSTEM — EXAKT — KEINE ABWEICHUNGEN

### CSS-Variablen (in index.css definieren):
```css
:root {
  --black:  #0a0704;
  --amber:  #e8991a;
  --amber2: #c97d10;
  --warm:   #f5e8c8;
  --dim:    rgba(245,232,200,0.72);
  --dimmer: rgba(245,232,200,0.55);
}
```

### Hintergrund & Basis:
- `background: var(--black)` — tiefschwarz mit leichtem Braunstich
- `color: var(--warm)` — warmes Crème
- `font-family: 'Lora', Georgia, serif` — Body
- KEIN Lila, KEIN Magenta, KEIN Violett, KEINE blauen Töne

### Typografie:
- Headlines: `font-family: 'Playfair Display', serif` — fett, manchmal kursiv
- Body: `font-family: 'Lora', serif` — lesbar, warm
- Akzente: `color: var(--amber)` — warmes Gold
- Fließtext: `color: rgba(245,232,200,0.78)` — leicht gedimmt, aber gut lesbar
- Labels / Nebentexte: MINDESTENS `rgba(245,232,200,0.55)` — nie dunkler

### Buttons:
- Primary: `border: 1px solid var(--amber); color: var(--amber); background: transparent`
- Hover: `background: var(--amber); color: var(--black)`
- Font: Playfair Display, kursiv

### Inputs / Textarea / Select:
- `background: rgba(245,232,200,0.04)`
- `border: 1px solid rgba(245,232,200,0.12)`
- `color: var(--warm)`
- Focus: `border-color: rgba(232,153,26,0.55)`
- Placeholder: `color: rgba(245,232,200,0.45)` — gut lesbar

### Scroll Reveal:
- Hook `useReveal` mit IntersectionObserver
- Klasse `.reveal` → opacity 0, translateY 16px → visible: opacity 1, translateY 0
- Transition: 0.7s ease
- Delay-Klassen: `.d1` (0.1s), `.d2` (0.2s), `.d3` (0.3s)

```typescript
export const interessenten = pgTable('interessenten', {
  id: serial('id').primaryKey(),
  name: text('name').notNull(),
  personen: text('personen').notNull(),
  statement: text('statement'),
```

## 8. DATENBANK-SETUP

```typescript
// src/lib/db.ts
import { drizzle } from 'drizzle-orm/node-postgres';
import { Pool } from 'pg';
import * as schema from './schema';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });
export const db = drizzle(pool, { schema });
```

```typescript
// src/lib/schema.ts
import { pgTable, serial, text, timestamp } from 'drizzle-orm/pg-core';

export const interessenten = pgTable('interessenten', {
  id: serial('id').primaryKey(),
  name: text('name').notNull(),
  personen: text('personen').notNull().default('1'),
  statement: text('statement'),
  song: text('song'),
  createdAt: timestamp('created_at').defaultNow(),
});
```

Migration ausführen mit: `drizzle-kit push`

## 9. API-ROUTE

```typescript
// POST /api/interesse
// Request body: { name: string, personen: string, statement?: string, song?: string }
// Response: { success: true } | { error: string }

// Validierung mit Zod:
const schema = z.object({
  name: z.string().min(1).max(100),
  personen: z.string(),
  statement: z.string().max(500).optional(),
  song: z.string().max(200).optional(),
});
```
