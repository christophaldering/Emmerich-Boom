## TEIL 2 — Kino-Modus (in `artifacts/emmerich/src/pages/ThekePage.tsx`)

Ein Schalter, der alles ausblendet außer der Bilderwand (für Hoch- und besonders Querformat). Auf Laptop/Android zusätzlich echtes Vollbild; auf iPhone wird unsere eigene Oberfläche ausgeblendet (Vollbild-API ignoriert iOS still).

### 2a. State + Handler (in der Haupt-Theke-Komponente, die die Szene rendert)

```tsx
const [kino, setKino] = useState(false);

const enterKino = () => {
  setKino(true);
  const el = document.documentElement as any;
  const p = el.requestFullscreen?.();        // iOS: undefined → kein Aufruf
  if (p && typeof p.catch === "function") p.catch(() => {});
};
const exitKino = () => {
  setKino(false);
  if (document.fullscreenElement) (document as any).exitFullscreen?.();
};

useEffect(() => {
  const onKey = (e: KeyboardEvent) => { if (e.key === "Escape") setKino(false); };
  const onFs = () => { if (!document.fullscreenElement && document.fullscreenEnabled) setKino(false); };
  window.addEventListener("keydown", onKey);
  document.addEventListener("fullscreenchange", onFs);
  return () => { window.removeEventListener("keydown", onKey); document.removeEventListener("fullscreenchange", onFs); };
}, []);
```

**Wichtig:** `requestFullscreen` MUSS wie oben gegen `undefined` abgesichert sein (sonst Absturz auf iPhone).

### 2b. Im Render: bei `kino === true` ausblenden

Folgende vorhandene Blöcke nur rendern, wenn **nicht** im Kino-Modus — jeweils mit `{!kino && ( … )}` umschließen:

- die **Grußzeile** (`{/* ── Grußzeile … ── */}`)
- die **Tresen-Region** (`{/* ── Tresen-Region … ── */}`)
- die **Phasen-Andeutungen** (`{/* ── Phasen-Andeutungen … ── */}`)
- der **„Zur Startseite”**-Link

### 2c. Wand-Region im Kino-Modus vergrößern

Die **Wand-Region** (`{/* ── Wand-Region: Galerie ── */}`) bleibt immer sichtbar, aber im Kino-Modus größer. Die Positionswerte abhängig von `kino` setzen:

- normal: wie bisher (`W.top/left/width/height`)
- kino: `top:"6%"`, `left:"3%"`, `width:"94%"`, `height:"82%"`

Backdrop (Bühnenbild) bleibt dahinter sichtbar. `GalerieWand` bleibt unverändert interaktiv (Wischen, Antippen, „▶ Abspielen” weiter nutzbar).

### 2d. Kino starten — kleiner Knopf oben rechts

Nur sichtbar, wenn **nicht** im Kino-Modus. Dezent, oben rechts, mit Safe-Area:

```tsx
{!kino && (
  <button onClick={enterKino} style={{
    position:"absolute", top:"max(0.9rem, env(safe-area-inset-top,0px))", right:"0.9rem", zIndex:11,
    background:"rgba(10,7,4,0.6)", border:`1px solid ${am(0.32)}`, borderRadius:"20px",
    padding:"0.3rem 0.8rem", fontFamily:"'Lora', serif", fontStyle:"italic",
    fontSize:"0.72rem", color:am(0.8), cursor:"pointer", backdropFilter:"blur(4px)",
  }}>⤢ Kino</button>
)}
```

### 2e. Kino verlassen — × oben rechts

Nur sichtbar, wenn im Kino-Modus:

```tsx
{kino && (
  <button onClick={exitKino} style={{
    position:"absolute", top:"max(0.9rem, env(safe-area-inset-top,0px))", right:"0.9rem", zIndex:11,
    background:"rgba(10,7,4,0.6)", border:`1px solid ${am(0.32)}`, borderRadius:"50%",
    width:"2.2rem", height:"2.2rem", color:am(0.8), fontSize:"1.3rem", lineHeight:1,
    cursor:"pointer", backdropFilter:"blur(4px)",
  }}>×</button>
)}
```
